# OpenVox Roadmap

> Community-driven voice dictation for Linux

## v1.1.0 - Enhanced Context (In Progress)

- [x] Session memory across dictations
- [ ] Multiple hotkey profiles (work/personal)
- [ ] Custom context shortcuts (email template, Slack DM, etc.)

## v1.2.0 - Platform Expansion

- [ ] macOS support
- [ ] Windows support (PowerToys integration)
- [ ] Mobile companion app (iOS/Android)

## v1.3.0 - AI Improvements

- [ ] Local Whisper fallback (faster, no API)
- [ ] Custom LLM endpoint support (Ollama, OpenAI)
- [ ] Context-aware rewriting with document history

## v2.0.0 - Major Features

- [ ] Multi-language dictation
- [ ] Voice commands (not just dictation)
- [ ] Plugin system for custom processors

## Backlog

- [ ] System tray menu for quick settings
- [ ] Recording quality presets (voice memo vs meeting)
- [ ] Export transcripts to markdown/docx
- [ ] Keyboard shortcut customization UI
- [ ] Dark mode for web dashboard
- [ ] PWA support for mobile web

## How to Request Features

1. Open a GitHub Discussion
2. Describe your use case
3. Explain why existing features don't work
